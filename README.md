# Wind Farm Analytics Opportunity Assessment

## Overview

This repository contains an exploratory analysis of wind turbine operational data from the Penmanshiel Wind Farm dataset.

The aim of the investigation is to identify data-driven opportunities to improve wind farm performance, reliability, and availability through scalable analytics solutions. Rather than focusing on a single failure event, the analysis evaluates how operational data can be used to develop reusable monitoring and predictive maintenance capabilities applicable across multiple wind farms.

## Objectives

- Assess the quality and structure of available SCADA and event data.
- Identify patterns associated with asset performance degradation and turbine downtime.
- Explore fleet-wide benchmarking and anomaly detection approaches.
- Evaluate opportunities for predictive maintenance and early fault detection.
- Propose analytics products that can be generalised across different wind farm deployments.

## Dataset

The analysis uses the publicly available Penmanshiel Wind Farm dataset, which contains:

- Wind turbine SCADA measurements
- Event and alarm records
- Site-level operational data
- Turbine metadata

Source:
https://zenodo.org/records/5946808

## Approach

The investigation follows four stages:

1. Data ingestion and quality assessment
2. Exploratory analysis of turbine performance and failures
3. Development of candidate health and reliability indicators
4. Evaluation of scalable analytics opportunities

## Deliverables

- Data quality assessment
- Exploratory analysis notebooks
- Visualisations and findings
- Recommendations for deployable analytics solutions
- Discussion of implementation trade-offs and future work

## Disclaimer

This work was completed as part of a technical interview exercise and is intended to demonstrate analytical methodology, problem-solving approach, and product-oriented thinking rather than provide a production-ready solution.
